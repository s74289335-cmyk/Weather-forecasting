# 🌦️ Global Weather Trend Forecasting

## 📌 Overview

This project analyzes global weather data and forecasts future temperature trends using advanced data science and machine learning techniques.
It combines statistical and deep learning approaches to provide accurate and interpretable weather predictions.

---

## 🚀 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Advanced Visualizations
* Anomaly Detection (Isolation Forest)
* Climate Trend Analysis
* Environmental Impact Analysis (Air Quality vs Temperature)
* Feature Importance Analysis
* Forecasting Models:

  * ARIMA
  * Prophet
  * Ensemble Model
* Interactive React Dashboard (Location-based forecasting)

---

## 📊 Models Used

* ARIMA (Time Series Forecasting)
* Prophet (Trend & Seasonality Modeling)
* Ensemble Model (Improved Accuracy)

---

## 📈 Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)

---

## 🌍 Insights

* Temperature trends vary across different geographical locations
* Air quality indicators (PM2.5, CO) influence temperature patterns
* Seasonal variations are observed in long-term climate analysis
* Anomalies highlight extreme weather events

---

## ⚙️ How to Run

### 🔹 Backend

cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### 🔹 Frontend

cd frontend
npm install
npm start

---

## 📊 API Endpoints

* `/locations` → Get available locations
* `/forecast/{location}` → Get forecast for selected location
